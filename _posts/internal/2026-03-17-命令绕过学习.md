如果cat被禁
sed -n '1,10p' /etc/passwd

打印整个文件
sed -n 'p' /etc/passwd

只看某一行
sed -n '5p' /etc/passwd

ps aux | sed -n '1,5p' 看前5行进程

sed 's/a/b/g' file #替换
sed -n 'p' file #打印文件
sed -n '1,10p' file #指定行打印
sed -i 's/a/b/g' file #修改文件

如果cat/ls被禁  立刻想到sed/more/less/awk

more <(sed -n 'p' /etc/passwd)

less <(awk '{print}' /etc/passwd)

还有 less <(sed '' /etc/passwd)

less <(/bin/cat /etc/passwd)

那如果不能用cat / less / more /sed

awk '{print}' /etc/passwd
awk -F: '{print $1}' /etc/passwd
awk -F: '/root/ {print}' /etc/passwd 匹配包含root的行
awk 'NR \==1,NR \==10' /etc/passwd 

如果awk也不能用

while read line; do echo $line; done < /etc/passwd

for i in $(cat /etc/passwd); do echo $i; done

不用cat 的版本
for i in $(</etc/passwd); do echo $i; done

echo "$(< /etc/passwd)"
